# Data pack feature matrix

[Back to README](../README.md)

This page tracks which data pack provides each data-driven gameplay or world behavior feature in released Minecraft versions. Sections group related goals, rows describe specific features, and version columns name the data pack used for that version.

## Global data packs

These entries are distributed through the pack-level datapacks directory and loaded globally by the selected global pack loader.
