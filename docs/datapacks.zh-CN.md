# 数据包功能矩阵

这页跟踪每个已发布 Minecraft 版本中，由哪个数据包提供对应的数据驱动玩法或世界行为功能。章节用于归类相关目标，表格行描述具体功能，版本列填写该版本使用的数据包。

## 全局数据包

这些条目通过整合包层面的 datapacks 目录分发，并由当前选用的全局包加载器全局加载。
